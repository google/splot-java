com.google.iot.m2m.annotation.Property

Class Overview

Annotation describing the characteristics of a property in a trait. This annotation is applied to all PropertyKey fields designated as public static final in a trait class. The value of this annotation is a bit-mask describing the characteristics of the property.

Example


     @Property(READ_WRITE | GET_REQUIRED)
     public static final PropertyKey STAT_VALUE =
             new PropertyKey<>(
                     PropertyKey.SECTION_STATE,
                     TRAIT_ID,
                     "v",
                     java.lang.Float.class);

     @Property(CONSTANT)
     public static final PropertyKey META_NATIVE_MIREDS =
             new PropertyKey<>(
                     PropertyKey.SECTION_METADATA,
                     TRAIT_ID,
                     "mire",
                     java.lang.Float.class);
