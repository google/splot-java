Class Overview

Abstract class for more easily implementing local FunctionalEndpoints that support scenes.

By subclassing this class instead of LocalFunctionalEndpoint, scene support will be largely implemented for you automatically. Like LocalFunctionalEndpoint, Subclasses of this class should NOT be used to implement non-local FunctionalEndpoints: in those cases a class directly implementing the methods of FunctionalEndpoint should be used.

If scene support doesn't make sense for your functional endpoint (for example, if it is an input rather than an output), then you should subclass LocalFunctionalEndpoint instead.

If you need transition support in addition to scenes, use LocalTransitioningFunctionalEndpoint instead.

public Map<String, Object> copyPersistentState ()

Retrieves a copy of the state for the instance that should be saved to nonvolatile storage. When reconstructing the associated instance, call initWithPersistentState(Map) to restore this state.

The data contained in the returned Map should be considered opaque unless specified otherwise by the implementing technology.

The types for the values contained in the map are only guaranteed to be faithfully restoreable if they are one of the following types:

  • Integer
  • Double
  • Float
  • byte[]
  • String
  • Map<String,Object>, where Object is also one of these acceptable types
  • java.util.List<Object>, where Object is also one of these acceptable types
This allows the object to be easily serialized to CBOR.

This method MUST be thread safe.

Returns
  • the persistent state of this instance

public final Set<String> getSceneIds ()

Returns a the set of scene IDs that this functional endpoint currently has.

Note that this method is only present in LocalSceneFunctionalEndpoint, but you can get the same information with any other FunctionalEndpoint that supports SceneTrait by instead using the following more generic code: 

Collection<FunctionalEndpoint> sceneFeList = fetchChildrenForTrait(SceneTrait.PARAM_SCENE_ID).get();
 HashSet<String> sceneIdSet = new HashSet<>();
 for (FunctionalEndpoint sceneFe : sceneFeList) {
     sceneIdSet.add(getIdForChild(sceneFe));
 
 }
The above code will work for any FunctionalEndpoint that implements support for the scene trait, whereas this method will only be available for for subclasses of LocalSceneFunctionalEndpoint.

public void initWithPersistentState (Map<String, Object> persistentState)

Restores a persistent state previously retrieved from copyPersistentState(). This should be performed only once when this instance is being initialized. The behavior for calling this method more than once is undefined.

Parameters

public final void removeSceneId (String sceneId)

Removes the scene with the given sceneId from this functional endpoint.

Note that this method is only present in LocalSceneFunctionalEndpoint, but you can easily make the same call to other FunctionalEndpoints that support SceneTrait by instead using the following more generic code: 

FunctionalEndpoint sceneFe = this.getChild(SceneTrait.TRAIT_ID, sceneId);
 if (sceneFe != null) {
     sceneFe.delete().get();
 
 }
The above code will work for any FunctionalEndpoint that implements support for the scene trait, whereas this method will only be available for for subclasses of LocalSceneFunctionalEndpoint.

public final void saveCurrentStateToSceneId (String sceneId)

Saves the current state to the given scene id.

Note that this method is only present in LocalSceneFunctionalEndpoint, but you can easily make the same call to other FunctionalEndpoints that support SceneTrait by instead using the following more generic code: 

invokeMethod(SceneTrait.METHOD_SAVE, SceneTrait.PARAM_SCENE_ID, sceneId).get();
The above code will work for any FunctionalEndpoint that implements support for the scene trait, whereas this method will only be available for for subclasses of LocalSceneFunctionalEndpoint.
