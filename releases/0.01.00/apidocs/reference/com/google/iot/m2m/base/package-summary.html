package

com.google.iot.m2m.base

Splot base classes and interfaces.

Interfaces

ChildListener Interface for handling changes to a FunctionalEndpoint's children. 
ConfigListener Interface for handling changes to a FunctionalEndpoint's config properties. 
FunctionalEndpoint Interface for a Functional Endpoint. 
Group Interface for managing a group. 
LocalTrait Interface which is implemented by helper classes (like BaseTrait.AbstractLocalTrait) which are generated exclusively by the trait annotation processor. 
MetadataListener Interface for handling changes to a FunctionalEndpoint's metadata properties. 
PersistentStateInterface Interface for enabling the persistent storage of information related to an object. 
PersistentStateListener Interface used to receive updates from a PersistentStateInterface regarding the need to refresh the data stored in non-volatile memory. 
PersistentStateManager A persistent state manager handles the retrieval and updating of the persistent state of objects which implement PersistentStateInterface. 
PropertyListener<T> Interface for receiving asynchronous notifications for changes to individual properties on a FunctionalEndpoint. 
StateListener Interface for notifications of changes to a FunctionalEndpoint's state properties. 
Technology Interface for discovering, interacting with, and hosting FunctionalEndpoints. 

Classes

DiscoveryBuilder Builder class for constructing a new DiscoveryQuery object. 
DiscoveryQuery An abstract class representing a specific functional endpoint discovery query. 
DiscoveryQuery.Listener Callback class for handling functional endpoints immediately as they are discovered and being notified of the completion of a DiscoveryQuery. 
MethodKey<T> A generic class for identifying and facilitating return-type-safety for methods. 
ParamKey<T> A generic class for identifying and facilitating return-type-safety for method parameters. 
PropertyKey<T> A generic class for facilitating type-safety on individual properties while remaining convenient to use. 
TypedKey<T> Contains the shared implementation of PropertyKey, MethodKey, and ParamKey.

CorruptPersistentStateException Checked exception for when the underlying persistent state data is corrupted. 
GroupNotAvailableException Checked exception which is thrown if one of the group-management methods of Group are called after the group has been deleted or otherwise no longer available. 
GroupsNotSupportedException Checked exception indicating that this Technology does not support groups. 
InvalidMethodArgumentsException Checked exception thrown when the arguments to a Splot method are invalid. 
InvalidPropertyValueException Checked exception thrown after an attempt to set a property to an invalid value. 
InvalidValueException A checked exception specific to TypedKey that indicates that a cast or coercion has failed. 
MethodException Checked exception indicating problems and errors related to invoking methods. 
MethodNotFoundException Checked exception that indicates that a given method could not be found on this FunctionalEndpoint. 
PropertyException Checked exception indicating problems and errors related to the changing or fetching of properties on FunctionalEndpoints. 
PropertyNotFoundException Checked exception that indicates that a given property could not be found on this FunctionalEndpoint. 
PropertyOperationUnsupportedException Checked exception that indicates that the operation being performed is not available for the given property on this FunctionalEndpoint. 
PropertyReadOnlyException Checked exception that indicates that a given property can only be read, not written. 
PropertyWriteOnlyException Checked exception that indicates that a given property can only be written, not read. 
TechnologyCannotHostException Checked exception thrown when a given Technology instance rejects a FunctionalEndpoint argument because it is somehow unacceptable. 
TechnologyException Checked exception thrown by Technology instances (and their native FunctionalEndpoint instances) indicating a technology-specific error has occurred. 
TechnologyRuntimeException Unchecked exception thrown by Technology instances (and their native FunctionalEndpoint instances) indicating a serious technology-specific runtime error has occurred.
